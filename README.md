# BriefLink

BriefLink helps operators and founders turn scattered company information into a shareable account brief they can send before meetings, outreach, and handoffs.

## Workspace layout
- `workspace/frontend/` — frontend dept owns application UI and prototype implementation
- `workspace/backend/` — backend dept owns API and server logic
- `workspace/devops/` — devops dept owns deployment and runtime planning
- `workspace/growth/` — growth dept owns launch and acquisition assets
- `workspace/vizops/` — vizops dept owns visualization artifacts and readiness views
- `workspace/docs/` — platform-owned shared operating docs and cross-team setup notes
