# BriefLink Source Bundle v1

## Usage notes
This is the consolidated BriefLink source bundle for launch-phase growth execution. It packages finalized source assets in one document and applies the locked microcopy v1 standard plus the approved voice/UI language baseline.

### Locked microcopy v1 rules
- Primary pre-signup CTA: `Get early access`
- Post-signup confirmation only: `You’re on the list`
- Required terminology: `account risk`
- Preferred phrasing: `before client calls`
- Required line: `Prepare with a clear brief.`
- Message wedge: calm client prep + risk visibility
- Audience: profitable, reachable agencies and consultancies; especially agency owners, principals, account directors, client partners, consultancy leaders, and client-service leads
- Do not drift into generic SMB productivity, approval-cycle framing, or feedback-chain framing

### Approved voice/UI language pack
- Tone: calm, credible, operational, client-facing
- Voice: clear, low-hype, specific, steady
- UI language: direct labels, plain-English actions, no novelty phrasing, no soft/ambiguous CTA wording
- Buttons and submits should use action language already approved in microcopy v1
- Confirmation states should be concise and factual
