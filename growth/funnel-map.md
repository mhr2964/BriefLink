# BriefLink funnel map

| Stage | Audience | Channel | Asset reference | Goal |
|---|---|---|---|---|
| Awareness | Agency / consultancy founder or operator | Founder email | founder-outreach.md / Email 1 | Establish relevance |
| Consideration | Agency / consultancy founder or operator | Founder email | founder-outreach.md / Email 2 | Clarify problem and value |
| Reminder | Agency / consultancy founder or operator | DM | founder-outreach.md / DM follow-up | Light-touch follow-up |
| Close-loop | Agency / consultancy founder or operator | Founder email | founder-outreach.md / Email 3 | Prompt reply or disqualify |
| Waitlist | Interested early user | Email | first-user-lifecycle.md / Waitlist confirmation | Confirm signup and set expectations |
| Onboarding | New user | Email + in-app | first-user-lifecycle.md / Welcome | Drive first real use |
| Activation | Inactive new user | Email + SMS/in-app | first-user-lifecycle.md / Activation nudge | Trigger first evaluation |
| Early retention | Light-usage user | Email + in-app | first-user-lifecycle.md / Value reminder | Reinforce fit |
| Feedback | Active early user | Email + in-app | first-user-lifecycle.md / Feedback request | Collect insight |
| Referral | Satisfied early user | Email + SMS/in-app | first-user-lifecycle.md / Referral / invite ask | Generate warm referrals |
| Reactivation | Inactive prior user | Email + SMS/in-app | first-user-lifecycle.md / Reactivation | Recover stalled users |
